# AI Learning 🤖

AI 技能学习笔记与实践项目，涵盖机器学习、深度学习、LLM、Agent 等方向。

## 📚 学习路线

### 第一阶段：基础入门
- Python 数据科学基础（NumPy、Pandas、Matplotlib）
- 机器学习基础概念与经典算法
- 动手实践：Scikit-learn 入门项目

### 第二阶段：深度学习
- 神经网络基础（前馈网络、CNN、RNN）
- PyTorch / TensorFlow 框架实践
- 计算机视觉与自然语言处理入门

### 第三阶段：大语言模型（LLM）
- Transformer 架构原理
- Prompt Engineering 技巧
- LLM API 调用与应用开发（OpenAI、Claude、通义千问等）
- RAG（检索增强生成）实践
- 微调（Fine-tuning）入门

### 第四阶段：AI Agent
- Agent 架构与设计模式
- LangChain / LlamaIndex 框架
- Function Calling 与工具使用
- 多 Agent 协作系统
- MCP（Model Context Protocol）实践

### 第五阶段：项目实战
- 搭建个人 AI 助手
- RAG 知识库问答系统
- AI 编程助手工作流
- 多模态应用探索
